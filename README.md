# Travelic - 여행의 즐거움
## [🔗URL](https://hanbinchoi.github.io/Travelic/index.html)
**Travelic은 대륙 별 여행도시를 소개하고 먹거리, 가볼만한 곳의 정보를 알려줍니다. 순수 JavaScript로 구현되었으며 CSS, HTML 을 사용하였습니다. 또한 지도 표시를 위해 Google Map API를 사용합니다. Menu bar, title animation, scroll animation, 화살표 click 이벤트 등이 JS를 통해 구현되었습니다.**
<br>
<br>

## 🌏대륙 페이지(Asia)


#### 대륙 사이트는 메인 타이틀 section과 각 대륙 별 추천도시 section으로 나뉘어져 있습니다

<img src="https://user-images.githubusercontent.com/44600584/139659639-677c078a-1a68-4ab5-ad75-2934015da783.gif"  width="450" height="250"/>

##### 메인 타이틀 section은 typing text 효과와 그에 맞추어 이미지가 전환되고 에니메이션이 끝나면 filter가 적용됩니다.

<img src="https://user-images.githubusercontent.com/44600584/139659914-849b6e9e-8c60-4ceb-81bd-f654306cbaf6.gif"  width="450" height="250"/>

##### header는 스크롤과 동시에 사라지면서 왼쪽에 메뉴바가 생성됩니다.

<img src="https://user-images.githubusercontent.com/44600584/139660014-e0a8d2d4-c329-4536-8704-e7d959a210bd.gif"  width="450" height="250"/>

##### 추천도시 section은 추천도시가 card flip으로 띄워져 있으며 마우스를 올리면 해당 도시의 정보를 나타냅니다. 클릭 시에는 해당 도시 페이지로 이동합니다.

## 🏙도시 페이지(Osaka)

#### 도시 페이지는 스크롤 animation section, 해당 도시 정보 section, 먹거리, 가볼만한 곳 section으로 구성되었습니다.

<img src="https://user-images.githubusercontent.com/44600584/139661633-a66a4ec1-f352-4520-8310-2dbbac2bd9d2.gif"  width="450" height="250"/>
